My Projects

LRA+

I developed a comprehensive Discord bot named LRA+ using the Nextcord library, an improved version of Discord.py, designed to interact with the Discord API. I configured the bot with extensive functionalities including role assignment, a reputation system, invite tracking, games, shops, and image generation using OpenAI’s API. Initially, the bot loads configuration from a file and sets up intents for guilds, message content, presences, and members. It creates a bot instance with a specified command prefix and defines several global dictionaries to manage invites, reputation, diamonds, and last claimed timestamps. Upon starting, the bot loads data from persistent storage, refreshes interactive messages, and sets up periodic tasks for refreshing messages, backing up data, and updating reputation. The bot handles various events such as member updates, invite creation and deletion, member joins, and member removals, updating reputation points and sending appropriate notifications.

Role assignment is managed through an interactive view with buttons, enabling users to add or remove specific roles. The bot includes commands for clearing messages, checking its latency, sending messages to specific channels, rebooting, adding reputation manually, and saving reputation data. I implemented a detailed reputation and ranking system, allowing for automated reputation updates based on voice channel activity and periodic checks for rank advancements. The bot features multiple games, including a coin flip game, a daily lottery, and a blackjack game with detailed handling of game states, betting, and result processing. Shops are implemented where users can buy roles and custom colors using reputation points, trade diamonds for reputation, and purchase premium roles using diamonds.

The bot also integrates OpenAI's API for generating images based on user prompts, with reputation points deducted for each generation. Additionally, the bot provides detailed statistics on various activities, including games, daily claims, shop transactions, and AI interactions, updating a designated message periodically with the latest stats. Backup tasks ensure data is regularly saved, and a music playback feature allows users to play, pause, resume, skip, and stop music in voice channels. The bot’s error handling ensures smooth operation, notifying users of issues and providing a user-friendly experience across its multiple functionalities. I efficiently managed the testing and debugging processes, ensuring that the bot now operates smoothly without any bugs or errors. Additionally, I was responsible for creating and hosting the bot using a virtual machine running Ubuntu in Google Cloud.
