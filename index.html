LRA+ Discord Bot

A bot that performs various tasks on Discord servers.

  • User Interaction & Management:
    • Role management via interactive buttons.
    • Automated welcome/farewell messages.
    • Commands for clearing messages, checking latency, sending messages, and bot restarts.
  • Reputation & Rewards System:
    • Automated reputation tracking for user activities.
    • Daily rewards with random chances.
  • Games & Entertainment:
    • Blackjack Game: Betting, scoring, and dealer play.
    • Lotto Game: Number guessing for reputation rewards.
    • Flip Coin Event: Randomized coin flips for user engagement.
  • Music Playback:
    • Music streaming from YouTube using yt_dlp.
    • Commands for playing, queuing, pausing, resuming, and stopping music.
  • AI Integration:
    • OpenAI GPT-powered AI assistant for user queries.
    • DALL-E model for image generation.
  • Statistics & Analytics:
    • Tracking and displaying various user and server activities.
    • Regular updates to a dedicated statistics channel.
  • Shop System:
    • Purchase roles and customizations using internal currency.
    • Premium upgrades for high-level users.
  • Backup & Data Management:
    • Regular backups of user data to prevent loss.
  • Cloud Hosting:
    • Deployed and managed the bot on Google Cloud Platform (GCP).
